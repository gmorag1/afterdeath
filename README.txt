HOW TO USE:
===========

1. Open index.html in any modern web browser
2. No server required — works as a standalone file
3. All styling is inline CSS
4. Tailwind CSS imported from CDN
5. No external dependencies beyond Tailwind and Google Analytics

TO CUSTOMIZE:
=============

1. Replace GA4 ID: Search for "G-XXXXXXXXXX" and replace with your actual ID
2. Update contact email: Change "hello@tendwell.com" to your email
3. Add/modify providers: Edit the provider-card divs in the directory section
4. Adjust colors: Modify the CSS color values
5. Add additional sections: Follow the existing pattern
